## 3. TrainerDeskの差別化ポイント

### 強み
1. **日本市場特化**
   - 完全日本語対応
   - チョコザップ等の日本のジムマシンをプリセット
   - 日本の食事文化に対応

2. **シンプルなUX**
   - 役割切り替え（スワイプ）でトレーナー/トレーニーを簡単に切り替え
   - トレーナー自身のセルフマネジメントも可能

3. **ガイド付きトレーニング実行**
   - テンプレートを選んで順番にガイド
   - 進捗バー表示で完了まで導く

4. **無料で始められる**
   - Firebaseの無料枠で小規模運用可能
   - 将来的な課金機能は段階的に追加

### 競合に対する課題（v1での未実装）
| 機能 | 対応予定 | 備考 |
|------|----------|------|
| 決済機能 | v2 | Stripe連携 |
| ビデオデモ | v2 | エクササイズ動画 |
| 複数トレーナー対応 | v2 | ジム向け |
| カスタムブランディング | v3 | ホワイトラベル |

### 検討中の機能
| 機能 | 対応予定 | 備考 |
|------|----------|------|
| 自動メニュー生成 | v2〜v3 | トレーナーの補助として。最終判断はトレーナー |

### 意図的に実装しない機能
| 機能 | 理由 |
|------|------|
| AI栄養分析 | 栄養士の専門領域。トレーナーの判断範囲外 |
